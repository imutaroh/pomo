# Pomo 市場性判定（2026-06-11）

## 最終判定: 売れる

**Claude としての最終判断（2026-06-11、製品側ブロッカー解消後に確定）: 推奨戦略（$19 買い切り・英語圏・Paddle 直販）で出せば、Pomo は売れる。**

- 「売れる」= 売上が立つこと。これは高確度。同一の開発者層・同価格帯で LookAway が 4,000 本売った実証テンプレートがあり、Pomo には実在する固有の差別化（フローモード × 全Space/フルスクリーン追従 × Claude Code 連携 API — この組み合わせの既存品は競合調査で確認できなかった）がある。下振れシナリオ（ローンチ不発）でも数十本は出る価格・チャネル設計
- 「儲かる」とは区別する。初年度中央値は ¥15万〜80万、期待値 約¥50万。時給換算では割に合わない。価値の本体は売上＋Swift 出荷経験＋ポートフォリオの合算
- 判定を覆す条件も明記する: 需要検証（デモ動画）の反応がゼロ署名なら撤退基準どおり無料/OSS 路線へ転換する。それでもこの判定（=製品と戦略の妥当性）は「売り方を変える」だけで無価値にはならない
- 製品側の出荷ブロッカーは解消済み: ✅ API トークン認証（実装済み）✅ 会議・通話中のオーバーレイ自動延期（実装済み）。残作業は売り方の実務（Xcode化・公証・Paddle・英語化）のみで、技術的な未知はない

## 推奨戦略

| 項目 | 推奨 | 根拠 |
|------|------|------|
| 価格 | **US$19 買い切り**（¥2,980）、14日トライアル | LookAway の実証価格。¥300帯は monora が実証した死路。サブスクは無料競合と正面衝突 |

## 売る前の絶対条件（P0）

1. ✅ **API トークン認証**（2026-06-11 実装済み。`~/Library/Application Support/Pomo/token`、Bearer 認証）
2. ✅ **会議・通話中のオーバーレイ自動延期**（2026-06-11 実装済み。CoreAudio マイク監視、権限プロンプト不要、トグルでOFF可）
3. **需要の事前検証**: Claude Code 連携のデモ動画を X/HN に出す。反応が閾値未達なら販売を中止し「無料/OSS + ポートフォリオ」路線へ転換する**撤退基準を先に決める**
4. Xcode プロジェクト化 + Developer ID 署名 + notarytool 公証（Apple Developer Program $99/年 — ご主人様の支出判断が必要な唯一の項目）
5. Paddle 開設 + ライセンスキー + トライアルロジック
6. Sparkle 自動更新
7. UI 英語化 + 英語 LP + デモ動画

## 技術判定

- **DB は導入しない（JSONL 維持で確定）**: 年1.4MB・追記専用・集計は数ms。BigQuery のネイティブロード形式が newline-delimited JSON なので、現ファイルがそのままエクスポート成果物。SQLite/SwiftData/Core Data は全部この規模では過剰。移行トリガー: 50万行超 or パース300ms超 or 複数プロセス書込 or 全文検索が必要になったら GRDB 一択
- **MAS 化の地雷**: サンドボックスで sessions.jsonl が `~/Library/Containers/` 配下に移動し、外部からの直読みが壊れる → 連携は HTTP API 経由に一本化してから

## この判定が破綻しうる条件

- 需要検証（デモ動画の反応）が閾値未達 → 即「無料/OSS」路線へ。この場合も Swift/出荷経験という投資回収は成立する
- Claude Code 連携の訴求が6ヶ月以内に模倣される（OSS 競合が API を付ける）→ 先行者の窓は狭い。出すなら早く
- 英語圏ローンチに必要な英語での発信・サポートのコストを負えない場合、中央値はさらに下がる
