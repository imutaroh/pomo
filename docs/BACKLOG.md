# Pomo ブラッシュアップ・バックログ

2026-06-11 の調査ワークフロー（競合監査・AI連携・休憩UX・統計UX の4軸 + 現実装のデザイン批評、エージェント6体）の統合結果。

## 反映済み（2026-06-11）

- ✅ justFinished グローが休憩中ずっと残るバグ → 6秒で自動消灯
- ✅ グローのハロー光が clipShape で削られる → クリップ外の overlay + 12px 余白
- ✅ idle 時にボタンが見えない初回体験 → 待機中はコントロール常時表示、「開始待ち」→「いつでもどうぞ」
- ✅ 貯めた休憩ピルをボタン化（報酬をクリックして受け取る構造）＋全ボタンにツールチップ
- ✅ メニュー刷新: 「透明度」→「集中時の濃さ」（逆読み防止）、モード別出し分け、説明的コピー
- ✅ 表示が嘘をつく3点: クラシック待機中は次の作業時間を予告表示 / フロー進捗バーは25分超で減光 / 1時間到達時のフォントジャンプ廃止（自動縮小）
- ✅ 休憩スキップに3秒クールダウン（one sec の PNAS 研究: 「自分で閉じる選択肢＋摩擦」が最も効果が持続。+5分と小さくは即時のまま）
- ✅ pomo CLI（未起動時の親切なエラー付き）＋ CLAUDE.md に API 仕様

## 次にやる（大きめの賭け）

| 項目 | 根拠 | 工数 |
|------|------|------|
| 統計の最小ラブラブル構成: 今日のメモ・タイムライン＋直近7日バー＋中立比較（「昨日: N」の裸の事実のみ、%・矢印・評価語なし） | 全アプリ調査の共通結論「毎日見るのは今日の合計と今日のログだけ」。memo 付き JSONL は interstitial journaling でこのアプリ最大の差別化資産 | L |
| サウンド制御（種類選択・音量・作業終了/休憩終了の別音、デフォルトは控えめに） | Flow 最大級の不満が「音量も種類も変えられない」。TomatoBar でも要望上位 | M |
| オーバーレイの事前警告（1分前にパネルの色温度変化 → フェードイン。突然出さない） | LookAway の知見「心の区切り」。クラシック/自動開始時は現状いきなり出る | M |
| 会議・画面共有・通話中のオーバーレイ自動延期（マイク/カメラ使用を監視） | このカテゴリ最多のクレームが「プレゼン中にオーバーレイが落ちてくる」。一度でも会議中に発火すると即アンインストール級 | L |
| README に信条を明文化（アカウントなし・テレメトリなし・サブスクなし・罪悪感なし） | TomatoBar は信条の明文化を信頼の感情価値に変えて 2,800★。「time blindness」という痛みの名指しが機能列挙より刺さる | S |
| macOS 集中モード（DND）連動（作業開始で ON、Shortcuts 経由なら private API 不要） | ブロッカーを作らずに集中を守れる唯一のミニマル整合な手段 | M |

## やらない（調査で根拠が確定）

| 項目 | 理由 |
|------|------|
| アカウント・クラウド同期・サブスク課金 | HN で最頻出の称賛が「アカウント不要」。Session は「機能は素晴らしいが価格が silly」と切り捨てられた。ローカルオンリー無料は欠点ではなく堀 |
| スキップ不能の強制ロック休憩（デフォルト） | Opal の事例: ユーザーは遵守ではなく回避策探しに走る。PNAS 研究でも最強要素は「自分で閉じる選択肢」 |
| ストリーク・ヒートマップ・ゲーミフィケーション | Forest の罪悪感メカニクスは「飽きる・長期効果弱い」が定着。ヒートマップの空白セルはストリークの変装。no-guilt 方針と正面衝突 |
| AI コーチ・割り込み検知・生産性スコア・ムード入力 | Centered（アプリ切替検知＋音声ナッジ）は「推測しすぎ・侮辱的」と酷評され倒産。判定・評価はストリークフリー思想と相反 |
| 単体 stdio MCP サーバー / URL スキーム | MCP はコンテキストを大量消費しエコシステムは CLI 回帰済み。必要になったら既存 APIServer に HTTP MCP を同居（Obsidian Local REST API 方式）で足りる。URL スキームは戻り値がなくエージェント I/O に不適 |
