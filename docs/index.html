はじめての開き方

現在は無料配布のため、初回だけ手動で開く必要があります。

  1. Pomo.dmg をダウンロードして開く
  2. Pomo を「アプリケーション」フォルダへドラッグ
  3. アプリケーションフォルダの Pomo を右クリック →「開く」→「開く」
  4. 「開発元を確認できません」と出たら：システム設定 → プライバシーとセキュリティ → 下の「このまま開く」
  5. 起動すると、画面右上のメニューバーに 🍅（Dockには出ません）

※ Apple の公証(notarize)を行っていないため、この手順が必要です。会社支給などの管理対象(MDM)Macでは起動できない場合があります。
