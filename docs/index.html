はじめての開き方

無料配布のため、初回だけ手動で開きます。

  1. Pomo.dmg を開き、Pomo を「アプリケーション」へドラッグ
  2. Pomo を右クリック →「開く」→「開く」
  3. 「開発元を確認できません」と出たら：システム設定 → プライバシーとセキュリティ → 下の「このまま開く」
  4. うまくいかないときは、ターミナルに1行：
    xattr -dr com.apple.quarantine /Applications/Pomo.app
  5. 起動すると、画面右上のメニューバーに 🍅（Dock には出ません）

Apple の公証(notarize)を行っていないための手順です。会社支給などの管理対象(MDM)Macでは起動できない場合があります。
